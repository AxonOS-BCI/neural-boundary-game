# API Surface

Reviewer-oriented helper functions in v1.9.1:

```text
release_ready()
boundary_status()
review_summary()
apply_script(actions)
risk_budget_remaining()
evidence_gap()
nbg_review_flags_after_demo_path()
nbg_release_blockers_after_demo_path()
```

These helpers keep the game useful as a deterministic protocol/review artifact, not only as a visual canvas.
