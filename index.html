    BOUNDARY PROTOCOL

    One membrane. Six decisions.

    Protocol reference. Opening this dialog pauses an active run.

    Private signal remains in the signal zone. Only validated, consented, evidence-backed typed intent may cross into the application zone.

    Validate
    Classify an unknown packet. False intent resolves to an artifact and still requires quarantine.
    Convert
    Transform validated intent only when scoped consent is active and evidence is at least L1.
    Quarantine
    Contain raw frames, artifacts, unsafe claims, and stimulation commands.
    Consent
    Activate a scoped capability or apply revocation immediately.
    Evidence
    Raise review maturity from trace to checksum to CI-confirmed proof.
    Release
    Seal only with trust ≥ 90, risk ≤ 20, integrity ≥ 80, L2 evidence, five review gates, valid conversion/release consent scopes, zero raw leaks, and no unresolved active packet.
