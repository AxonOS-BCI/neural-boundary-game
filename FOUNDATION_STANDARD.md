# AxonOS Foundation Standard — Boundary Console Public Surface

The public release candidate is positioned as a technical console, not a casual browser game.

## Public identity

**Name:** AxonOS Boundary Console  
**Version:** v7.3.0 RC1  
**Function:** deterministic Rust/WASM simulator for neural boundary systems  
**Core theme:** cognitive sovereignty, privacy vaults, typed intent, replayable evidence  

## What this repository demonstrates

- how raw neural signal can remain private;
- how applications can receive typed intent rather than unrestricted signal;
- how deterministic replay can support auditability;
- how boundary failures can be modeled as scenarios;
- how AxonOS can present safety-critical concepts through a controlled runtime surface.

## What this repository does not claim

- no medical-device certification;
- no clinical safety claim;
- no real neural data processing;
- no hardware stimulation control;
- no investment product;
- no custodial crypto flow.

## Public posture

The repository may be public, but it is a release candidate. The purpose is to show engineering seriousness, not to imply clinical readiness.
